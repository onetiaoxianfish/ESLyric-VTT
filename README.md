### VTT Lyrics Parser for ESLyric

将vtt字幕转为lrc



存在部分文件 是*.mp3.vtt 或者 *.wav.vtt

解决办法

点击左上角的文件 选择参数选项

工具一栏展开 ESLyric配置

点击歌词选项，在歌词来源内，选择本地文件一栏，点击右边配置

下方 搜索文件名 右键添加 %filename_ext% 这个，代表搜索文件名加后缀，也就是搜索到*.mp3 这种的文件名，这样就能正确匹配到 *.mp3.vtt 歌词后缀了
